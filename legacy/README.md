# Legacy Files

Folder ini menyimpan file proyek lama sebagai arsip referensi:

- `rekomendasi.ipynb`: notebook lama sebelum project dirapikan ulang.
- `rekomendasi.py`: hasil export script dari notebook lama.
- `recommender system.zip`: paket lama yang dipertahankan untuk jejak historis.

Pipeline utama project saat ini ada di:

- `src/book_recommender/`
- `notebooks/01_end_to_end_book_recommender.ipynb`
- `app/streamlit_app.py`

Gunakan file legacy hanya sebagai pembanding historis, bukan sebagai entrypoint
utama project.
